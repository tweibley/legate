# Contributing to ADK Ruby

## Getting Started
1. **Fork & Clone:** `git clone https://github.com/YOUR_USERNAME/adk-ruby.git`
2. **Setup:** Run `bin/setup` to install dependencies and check prerequisites.
3. **Verify:** Run `bundle exec rspec` to ensure everything works.

## Making Changes
*   **Branch:** `git checkout -b feat/my-feature` (or `fix/`, `docs/`)
*   **Test:** Add tests in `spec/`. Run `bundle exec rspec`.
*   **Lint:** Run `bundle exec rubocop` (and `-a` to fix) to ensure style compliance.
*   **Commit:** Use [Conventional Commits](https://www.conventionalcommits.org/) (e.g., `feat: add new tool`).

## Architecture
See **`AGENTS.md`** for detailed architecture and **`examples/`** for usage patterns.

## Submitting
Push your branch and open a Pull Request. Thank you for contributing!
