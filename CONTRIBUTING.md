# Contributing

1. **Setup**: Run `bin/setup` to install dependencies and configure `.env`.
2. **Test**: Run `bundle exec rspec` to ensure everything works.
3. **Style**: Run `bundle exec rubocop` before committing.

**PR Process**:
1. Fork & Clone
2. Create branch (`git checkout -b feature/xyz`)
3. Commit (`git commit -m "feat: description"`)
4. Push & Open PR

Questions? Check `README.md` or open an issue.
