# Contributing to ADK Ruby

We welcome contributions! 🚀

## Quick Start

1. **Setup**: Run `bin/setup` to install dependencies and configure your environment.
2. **Test**: Run `bundle exec rspec` to ensure everything works.
3. **Lint**: Run `bundle exec rubocop` to check code style.

## Development

- **Environment**: We use `dotenv` for configuration. `bin/setup` creates a `.env` from `.env.example`.
- **Testing**: Add tests for new features. We use RSpec.
- **Style**: We strictly follow RuboCop. Run `bundle exec rubocop -a` to fix issues.

## Pull Requests

1. Create a branch for your feature/fix.
2. Ensure tests and linting pass.
3. Open a PR with a clear description of changes.

Thank you for helping improve ADK Ruby!
