# Contributing

Thanks for contributing to ADK Ruby!

## Quick Start

1. **Fork & Clone**:
   `git clone https://github.com/YOUR_USERNAME/adk-ruby.git`
2. **Setup**:
   `bin/setup` (Installs gems, sets up .env, checks Redis)

## Development

- **Style**: Run `bundle exec rubocop` before committing.
- **Tests**: Run `bundle exec rspec` to ensure everything works.
- **Docs**: Update `README.md` for features. Agents see `AGENTS.md`.

## Pull Requests

1. **Branch**: `git checkout -b feat/my-feature`
2. **Commit**: Use [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat: ...`
   - `fix: ...`
   - `docs: ...`
3. **Push & PR**: Push to fork and open PR against `main`.

## Help

Open an issue on GitHub for questions.
