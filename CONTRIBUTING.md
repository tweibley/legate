# Contributing to ADK Ruby

Thank you for your interest! 🎉

## Getting Started

1. **Clone & Setup:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/adk-ruby.git
   cd adk-ruby
   bin/setup  # Installs deps & creates .env
   ```
2. **Create a branch:** `git checkout -b my-feature`

## Making Changes

### Code Style & Testing
We use RuboCop and RSpec. Please verify your changes:

```bash
bundle exec rubocop           # Check style
bundle exec rspec             # Run tests
```

### AI Agents
Refer to [AGENTS.md](AGENTS.md) for architectural guidelines.

## Submitting a Pull Request

1. **Push your branch:** `git push origin my-feature`
2. **Open a PR** against `main`.
3. **Wait for CI** to pass.

**Questions?** Open an issue on GitHub. Thanks for contributing! 🚀
