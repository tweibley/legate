# Contributing to ADK Ruby

Thank you for your interest in contributing! 🎉

## Getting Started

1. **Fork** the repository and clone your fork.
2. **Setup** the environment:
   ```bash
   bin/setup
   ```
3. **Branch** off `main` for your feature or fix.

## Making Changes

- **Code Style**: We use RuboCop.
  ```bash
  bundle exec rubocop
  ```
- **Testing**: All changes must include tests.
  ```bash
  bundle exec rspec
  ```
- **Documentation**: Update `README.md` or yard docs if needed.
  ```bash
  bundle exec yard
  ```

## Submitting a Pull Request

1. Ensure all tests pass and linting is clean.
2. Push your branch and open a PR against `main`.
3. Fill out the PR description with "What", "Why", and "Who".

## Getting Help

- Open an issue for questions or bugs.
- Check `README.md` and `AGENTS.md` for architectural details.
