# Contributing to ADK Ruby

We welcome contributions! Please fork the repo and create a PR.

## Quick Start

1. **Setup**:
   ```bash
   bin/setup
   ```
2. **Verify**:
   ```bash
   bundle exec rspec
   ```

## Guidelines

- **Architecture**: Read **[AGENTS.md](AGENTS.md)** first.
- **Style**: Run `bundle exec rubocop` (and `-a` to fix).
- **Tests**: Run `bundle exec rspec`. All changes must be tested.

## Submitting

1. Push your branch.
2. Open a PR against `main`.
3. Fill out the description.

**Questions?** Open an [issue](https://github.com/tweibley/adk-ruby/issues).
